Java előadás beadandó • Spring Boot & Forex

Forex kereskedési demonstrációs webalkalmazás

A projektünk célja egy oktatási célú, devizakereskedési webalkalmazás készítése, amely Spring Boot, Thymeleaf, az OANDA REST API és az MNB SOAP webszolgáltatás segítségével mutatja be a valós idejű és historikus devizaárfolyamok kezelését.

Az oldal megjelenítéséhez a Pico.css reszponzív, classless CSS keretrendszert használtuk, amely minimális HTML kóddal is modern, mobilbarát felületet biztosít.

Mit tud az alkalmazás?

Az alkalmazást több, egymásra épülő menüpont alkotja. Ezek együtt demonstrálják, hogyan tudjuk Java és Spring Boot segítségével külső pénzügyi szolgáltatásokhoz kapcsolódva a devizakereskedés alapvető folyamatait megvalósítani.

Főoldal menü

A Főoldal menü áttekintést ad az egész Forex demonstrációs webalkalmazásról, bemutatja a menüpontokat és a felhasznált technológiákat. Innen kiindulva érhető el az összes funkció.

Áttekintés, navigáció, technológiák rövid bemutatása

Vissza a főoldalra

SOAP menü

Az MNB SOAP webszolgáltatásából kérdezzük le a devizaárfolyamokat tetszőleges időintervallumra. Az eredményeket táblázatban és Chart.js grafikonon jelenítjük meg.

Technológiák: JAX-WS, SOAP, Chart.js

SOAP oldal megnyitása

Forex-account menü

Az OANDA számlaadatokat jelenítjük meg: egyenleg, realizált és nem realizált eredmény, valamint további számlainformációk.

Technológiák: OANDA v20 Java API, REST

Account megtekintése

Forex-AktÁr menü

Lenyíló listából kiválasztható egy instrumentum (pl. EUR_USD), majd az alkalmazás lekérdezi és kiírja az aktuális árfolyamot.

Technológiák: OANDA Pricing API, űrlapkezelés

Aktuális árak

Forex-HistÁr menü

Választható devizapár és idősűrűség (granularity), majd az alkalmazás lekéri az utolsó 10 historikus gyertyát, és táblázatban jeleníti meg.

Technológiák: OANDA InstrumentCandles, historikus adatok

Historikus árfolyamok

Forex-Nyit menü

Az űrlapon kiválasztható az instrumentum és a mennyiség. Pozitív mennyiség esetén Long, negatív esetén Short pozíciót nyitunk piaci áron.

Technológiák: MarketOrder, OANDA trade API

Pozíció nyitása

Forex-Poz menü

A Forex-Poz menüben a nyitott pozíciókat jelenítjük meg. A táblázatban jól áttekinthetően látható a trade azonosító, instrumentum, mennyiség és az aktuális eredmény.

Technológiák: trade.listOpen(), OANDA Java API

Nyitott pozíciók

Forex-Zár menü

A Forex-Zár menüben numerikus mezőben adhatjuk meg a lezárni kívánt tradeId-t, és a rendszer piaci áron lezárja az adott pozíciót.

Technológiák: trade.close(), OANDA Java API, űrlapkezelés

Pozíció zárása

Felhasznált technológiák

  • Java & Spring Boot – REST alapú webalkalmazás készítése
  • Thymeleaf – szerveroldali template motor a nézetekhez
  • OANDA v20 Java API – valós idejű Forex adat és trade kezelés
  • MNB SOAP webszolgáltatás – hivatalos devizaárfolyam lekérdezése
  • Chart.js – grafikonok rajzolása a SOAP menüben
  • Pico.css – modern, reszponzív, classless CSS téma

Az alkalmazás célja

A beadandóval bemutatjuk, hogyan tudunk egyetlen webalkalmazásban több külső szolgáltatást (SOAP + REST) integrálni, és ezekre építve egy egyszerű, de jól áttekinthető devizakereskedési demonstrációs rendszert kialakítani.

Használati javaslat

  1. Nyissuk meg a SOAP menüt, és vizsgáljuk meg az MNB árfolyamait.
  2. A Forex-account menüben ellenőrizzük a számlaadatokat.
  3. A Forex-AktÁr és Forex-HistÁr menükkel nézzük meg az aktuális és historikus OANDA árakat.
  4. A Forex-Nyit menüben hozzunk létre egy teszt pozíciót.
  5. A Forex-Poz menüben ellenőrizzük a nyitott pozíciókat, majd a Forex-Zár segítségével zárjuk le őket.
